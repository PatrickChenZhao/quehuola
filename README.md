# 缺货啦

一个面向奶茶店员工的手机横屏缺货记录原型。项目使用 Vite + React + TypeScript + Tailwind CSS 构建，纯前端运行，不包含登录、数据库、Excel 导出或本地持久化。

## 运行

```bash
npm install
npm run dev
```

打开终端提示的本地地址后，请使用手机横屏或浏览器移动设备横屏尺寸查看。

## 功能

- 竖屏提示横屏使用
- 横屏进入前台区、备料区、冰柜区场景
- 触摸左右滑动循环切换区域
- 点击物料选择或取消缺货
- 生成缺货表，按固定区域和摆放顺序展示
- 删除结果行会同步取消主界面选择
- 复制按区域分组的文字清单
